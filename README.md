# pretty-print-ast

> Formats ASTs as nice readable strings, with colors

## Usage Example

<!-- prettier-ignore -->
```ts
import { formatAst } from "pretty-print-ast";

const ast = /* get an AST from whatever */

  console.log(formatAst(ast));
```
